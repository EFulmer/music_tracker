About the Music Tracker

I'm Eric. I wrote this web app for two reasons:

  1. I have a bunch of friends with great music taste, and I wanted a neat way to keep track of everything they recommend me, without having to agonize over what to listen to.
  2. I love coding and learning more about how computers work. Before I started this project, I'd never done any web development.

I put 2 and 2 (or 1 and 2!) together, and built this app. Hopefully, you get some use out of it. :)

Boring technical stuff

I used Python because it feels very "lightweight" to me. My first languages were C++ and Java. Comparatively, Python has less overhead for getting stuff done. It's also got an amazingly huge variety of libraries. I think it's pretty neat.

Say hi!

This is my first web app, so I'm sure there's tons of stuff I could improve. If you see anything, say hi on Twitter; I'm @unironiceric.
